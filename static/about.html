What is this website?

This website is a visual representation of the backend blacklisting data being synced from MetaMask's eth-phishing-detect, 409h's EtherAddressLookup, and EtherScamDB.info.


Uses of this tool:

These blacklists are used to provide security for normal Ethereum users. By concatenating and syncing the blacklists of each project we provide a method to decrease the friction for between various ethereum scams being reported and their blacklisting being pushed out to end-user security tools.
